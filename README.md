# Nostr onboard

The simplest possible experience to onboard pre-nostr people to Nostr.

## Setup

```sh
yarn
```

To setup the build for native desktop and mobile platforms, more work is required. Documentation coming "soon".

## Start

```sh
yarn dev
yarn desktop
yarn android
```
